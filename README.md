# QA Banking Automation

End-to-end test automation framework for a banking application, built as part of a 26-week journey to become a QA Automation Engineer.

## Stack

- **Language:** Java 21
- **Test framework:** TestNG
- **UI automation:** Selenium WebDriver
- **API automation:** REST Assured, Postman
- **BDD:** Cucumber + Gherkin
- **Build:** Maven
- **Database validation:** SQL / MySQL
- **CI:** Jenkins
- **Version control:** Git, GitHub
- **Reporting:** Allure (planned)

## Demo Application

This framework targets [ParaBank](https://parabank.parasoft.com/) — a free public demo banking application by Parasoft.
